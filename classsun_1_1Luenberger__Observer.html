Detailed Description

Luenberger_Observer class: Implementation of Observer_Interface.

This class is a Discrete Luemberger Observer, it is also a Discrete SS system

 u(k) /-------\  y_hat(k)
 ---->|  Sys  |---->
 ---->|       |
 y(k) \-------/
        ^    |
x_hat(k)|    |x_hat(k-1)
        \----/

 x_hat(k) = f_obs(x_hat(k-1),u(k),y(k)) = f(x_hat(k-1),u(k)) + L * (y(k) - h(x_hat(k-1),u(k)))
 y_hat(k) = h(x_hat(k),u(k))

It stores the internal system state

A Discrete Luenberger Observer is also a Discrete state space system (SS_Interface). So, you can use the same interface. The only difference is the input that has to be redefined as:

 input = [observed_system_input; observed_system_output]

This is because the observer takes both the observed_system_input and observed_system_output as input.

In any case, you can use the specific observer methods: obs_state_fcn, obs_output_fcn, ... , that take as input the observed_system_input and observed_system_output separately.

See also
Linear_System_Interface, Kalman_Filter, Observer_Interface, Discrete_System_Interface
