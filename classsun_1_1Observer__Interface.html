Detailed Description

Observer_Interface class: represents a generic Discrete Observer.

This class is a generic discrete time Observer, it is also a SS system

 u(k) /-------\  y_hat(k)
 ---->|  Sys  |---->
 ---->|       |
 y(k) \-------/
        ^    |
x_hat(k)|    |x_hat(k-1)
        \----/

 x_hat(k) = f_obs(x_hat(k-1),u(k),y(k))
 y_hat(k) = h(x_hat(k),u(k))

It stores the internal system state

A Discrete observer is also a Discrete state space system (SS_Interface). So, you can use the same interface. The only difference is the input that has to be redefined as:

 input = [observed_system_input; observed_system_output]

This is because the observer takes both the observed_system_input and observed_system_output as input.

In any case, you can use the specific observer methods: obs_state_fcn, obs_output_fcn, ... , that take as input the observed_system_input and observed_system_output separately.

See also
Kalman_Filter, Luemberger_Observer, Discrete_System_Interface

Apply the observer, compute the estimated output and update the internal state.

Go one discrete step ahead, apply the input u(k) and the measure y(k), update the internal state for the next step, return the estimated output y_hat(k)

  x_hat(k) = f_obs(x_hat(k-1),u(k),y(k))
  y_hat(k) = h_obs(x(k),u(k))
Parameters
u_kObserved System Input at the current step u(k)
y_kObserved System Measure at the current step y(k)
Returns
estimated system output y_hat(k)
