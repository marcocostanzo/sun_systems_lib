Apply the system, compute the output and update the internal state.

Go one discrete step ahead, apply the input u(k), update the internal state for the next step, return the output y(k)

  x(k) = f(x(k-1),u(k))
  y(k) = h(x(k),u(k))

Apply the observer, compute the estimated output and update the internal state.

Go one discrete step ahead, apply the input u(k) and the measure y(k), update the internal state for the next step, return the estimated output y_hat(k)

  x_hat(k) = f_obs(x_hat(k-1),u(k),y(k))
  y_hat(k) = h_obs(x(k),u(k))
