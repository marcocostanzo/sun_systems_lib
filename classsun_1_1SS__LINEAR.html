Detailed Description

SS_LINEAR class: represents a Linear State Space System.

This class is a Linear State Space System:

u(k) /-------\  y(k)
---->|  Sys  |---->
     \-------/
       ^    |
   x(k)|    |x(k-1)
       \----/

x(k) = A*x(k-1) + B*u(k)
y(k) = C*x(k-1) + D*u(k)

It stores the internal system state

See also
Discretizator_Interface, RK4, Discrete_System_Interface

Apply the system, compute the output and update the internal state.

Go one discrete step ahead, apply the input u(k), update the internal state for the next step, return the output y(k)

  x(k) = f(x(k-1),u(k))
  y(k) = h(x(k),u(k))
