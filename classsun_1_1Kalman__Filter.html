Detailed Description

Kalman_Filter class: Implementation of Observer_Interface.

This class is a Discrete Extended Kalman Filter, Observer, it is also a Discrete SS system

 u(k) /----------------------------------\
 ---->|                                  |
 ---->|                                  |
 y(k) |                                  |  y_hat(k)
      |              EKF                 |---->
 W(k) |                                  |
 ---->|                                  |
 ---->|                                  |
 V(k) \----------------------------------/
        ^    |                  ^     |
x_hat(k)|    |x_hat(k-1)  P(k|k)|     |P(k-1|k-1)
        \----/                  \-----/

It stores the internal system state

A Discrete Discrete Extended Kalman Filter is also a Discrete state space system (SS_Interface).

Warning
This class is not fully implemented, it can be used only as a Kalman_Filter, do NOT try to use it as Observer_Interface or SS_Interface because the inheritance is NOT fully implemented.
Please, use ONLY the specific kf apply method: kf_apply, obs_apply.
Do not try to use the stateless *_fcn methods. Future implementations will handle this.

https://en.wikipedia.org/wiki/Extended_Kalman_filter

See also
Luenberger_Observer, Observer_Interface, Discrete_System_Interface

Apply the EKF, compute the estimated output and update the internal state.

Specific appy function for the EKF, is is similar to obs_apply(), but takes as input also the matrices W_k and V_k

It is equivalent to: 

  kf.setW(W_k);
  kf.setV(V_k);
  return obs_apply(u_k, y_k);

This method updates the internal state

Parameters

y_kObserved System Measure at the current step y(k)
W_kNew value for the W matrix
V_kNew value for the V matrix
Returns

Apply the observer, compute the estimated output and update the internal state.

Go one discrete step ahead, apply the input u(k) and the measure y(k), update the internal state for the next step, return the estimated output y_hat(k)

  x_hat(k) = f_obs(x_hat(k-1),u(k),y(k))
  y_hat(k) = h_obs(x(k),u(k))
