Detailed Description

Continuous_Observer_Interface class: represents a generic continuous Observer.

This class is a generic continuous observer interface.

A continuous observer is also a continuous state space system (Continuous_System_Interface). So, you can use the same interface. The only difference is the input that has to be redefined as:

input = [observed_system_input; observed_system_output]

This is because the observer takes both the observed_system_input and observed_system_output as input.

In any case, you can use the specific observer methods: obs_state_fcn, obs_output_fcn, ... , that take as input the observed_system_input and observed_system_output separately.

Note: This is a Continuous system, in order to simulate it you have to discretize it.

See also
Continuous_Luemberger_Observer, Continuous_System_Interface, Discretizator_Interface, RK4
